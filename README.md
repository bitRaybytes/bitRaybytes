## 🔐 HomeLab-Setup

Vollständig isolierte Angriffs- und Verteidigungsumgebung – aufgebaut, dokumentiert und kontinuierlich erweitert:

```text
    OSINT,                                   Angriff,
Reconaissance                               Exploits
+-----------+                             +------------+       +-----------------------------+
|   OPSec   |                             |  Hacking   |------>| Metasploitable2, DVWA, usw. |
+-----------+                             +------------+       +-----------------------------+
     ^                  Zugriff                 ^
     |                auf Internet              |
     |                  über FW                 |
+-----------+        +-----------+        +------------+
| Parrot OS | <----> |  pfSense  | <----> | Kali Linux |
+-----------+        +-----------+        +------------+
     ^                    ^                    ^
     |                    |                    |
     +--------------------+--------------------+
                          ^
                          |
+------------------------------------------------------+
|   Host OS (Windows) - Totale Isolation von VM        |
|   => Keine Gast-Erweiterungen aktiviert              |
+------------------------------------------------------+
```

**Was ich hier praktisch umgesetzt habe:**
- Port-Scanning und Reconnaissance mit **Nmap**
- Web-Traffic Analyse und Payload-Injektion mit **Burp Suite**
- Netzwerküberwachung mit **Wireshark**
- Simulierte Angriffe und Exploits mit **Metasploit** gegen Metasploitable 2, DVWA, SQLi usw.
- Grundlegende Firewall-Konfiguration und VLAN-Konzepte mit **pfSense**
- Alle Angriffe und Erkenntnisse dokumentiert im **Cybersecurity-Hub** *(privat – Einblick gerne auf Anfrage)*
